 Features
- **Smart Extraction:** Automatically identifies and extracts tables from multi-page PDFs.
- **Professional Formatting:** Applies custom styling, alternating row colors, and bold headers.
- **Summary Report:** Generates an overview sheet showing exactly what was processed.
- **Auto-Fit:** Dynamically adjusts Excel column widths for immediate readability.
🛠️ Installation
1. Clone the repository:
   `git clone https://github.com/AtulyaGupta1234567/pdf-to-excel-converter.git`
2. Install dependencies:
   `pip install -r requirements.txt`
 Usage
Place your PDF in the folder and run:
`python pdf_scraper.py your_file.pdf`
